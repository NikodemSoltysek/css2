Pojęcie Znaczenie pojęcie uzupełnia uczeń
Co znaczy skrót ID? ID oznacza "identifier" (identyfikator), czyli unikalną nazwę elementu w dokumencie HTML.
Jaki jest cel stosowania ID? ID służy do jednoznacznego wskazywania pojedynczego elementu na stronie, aby można było go stylować CSS lub odwoływać się do niego w JavaScript.
Jakie jest znaczenie znaku #? Znak # jest używany w CSS do odwołania się do elementu po jego ID, np. #nazwaID.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W CSS, ID wywołuje się za pomocą znaku #, np. #menu { color: red; }, co oznacza, że styl zostanie zastosowany do elementu o ID "menu".
Uwagi na temat stosowania ID (trzy konieczne wyliczenia).
  1. Każdy ID w dokumencie HTML musi być unikalny (niepowtarzalny).
  2. Stosuj ID tylko tam, gdzie naprawdę potrzebujesz jednoznacznego wskazania.
  3. W jednej stronie HTML nie powinno być dwóch elementów z tym samym ID.
